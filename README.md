[cciocov/livereload][1] is a [Docker][2] image running a [LiveReload][3] server.

### Usage

    $ docker run -d -p 35729:35729 -v /path/to/dir/to/watch/:/opt/watch/ cciocov/livereload

[2]: http://www.docker.com
[3]: http://www.livereload.com
